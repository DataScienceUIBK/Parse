به پرسش زیر با فهرستی از پاسخ‌های کوتاه و واقعی (factoid) که باید به زبان فارسی باشند پاسخ بده. ممکن است هیچ پاسخی وجود نداشته باشد یا یک تا چند پاسخ (حداکثر ۴ پاسخ) وجود داشته باشد. اگر پاسخی وجود ندارد، فهرست پاسخ‌ها را خالی بگذار.
پاسخ‌های فاکتوید کوتاه را فقط در قالب یک ساختار JSON زیر برگردان:

{
  "answers": [<پاسخ‌های کوتاه واقعی (حداکثر ۴ پاسخ کوتاه)>]
}

پرسش: دو پرنده که توانایی تقلید صدای انسان را دارند را نام ببرید.
پاسخ:
{
  "answers": ["طوطی", "مرغ مینا"]
}

پرسش: خواننده‌ای که در دهه ۴۰ خورشیدی موسیقی سنتی اجرا می‌کرد، در چه طرحی درباره آموزش ریاضی در تانزانیا همکاری داشته است؟
پاسخ:
{
  "answers": []
}

پرسش: نام کشوری که آمازون بزرگ‌ترین جنگل آن است چیست؟
پاسخ:
{
  "answers": ["برزیل"]
}

پرسش: چهار بخش از یک خانه سنتی ایرانی را نام ببرید که کارکردهای مختلفی دارند.
پاسخ:
{
  "answers": ["حیاط", "هشتی", "ایوان", "بادگیر"]
}

پرسش: کدام‌یک از واژگان رایج در زبان خیابانی تهران دارای ریشه ساختاری مشترک با زبان سومری هستند؟
پاسخ:
{
  "answers": []
}

پرسش: اگر احمد از رضا بزرگتر باشد و رضا از ناصر بزرگتر باشد، چه کسی از همه کوچکتر است؟
پاسخ:
{
  "answers": ["ناصر"]
}
